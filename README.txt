Planet Nibrek is a planet pack for KSP. It adds a planet called Nibrek, which shares orbit with Kerbin but is always on the other side of the sun. But the main bodies to be explored are Nibrek's two moons, Niblus and Nibort. Plus, Nibort is orbited by a small rock called Nibicus.

Dependencies:
Kopernicus.
Module Manager.
Modular Flight Integrator.

(Planet Nibrek will NOT work without these mods. Make sure you've installed them!)
